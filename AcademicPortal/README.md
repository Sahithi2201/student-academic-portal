## 📌 Project Overview
The Student Academic Portal is a web-based application developed to manage and access student academic information efficiently. The system allows students to view their marks and attendance, while administrators can manage student records through secure operations.

This project is implemented using Java in Eclipse IDE, with JDBC for database connectivity and Apache Tomcat as the web server.

---

## 🚀 Key Features

### 👩‍🎓 Student Module
- Secure Login Authentication
- View Academic Marks
- View Attendance Details

### 🛠️ Admin Module
- Add Student Records
- Update Student Marks
- Delete Student Records
- Manage Academic Data

## ⚙️ System Requirements
- Eclipse IDE
- Apache Tomcat Server
- MySQL Database
- Java JDK

---

## ▶️ How to Run the Project
1. Import the project into Eclipse IDE
2. Configure Apache Tomcat server in Eclipse
3. Setup MySQL database and update JDBC connection details
4. Run the project on the server
5. Access through browser (e.g., http://localhost:8080)
